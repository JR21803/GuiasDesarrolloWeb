Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.

Contenido:

Guía4

El contenido presentado en la cuarta guía se trata de una introducción al Responsive Design. Esto trata acerca de una técnica de diseño que permite a los sitios web a que se adapten según sea el dispositivo que se este utilizando para ver el contenido. Es por ello, que se hace uso de media queries que es lo que permite escribir estilos que respondan a los cambios de pantalla.

  • Utilizando los media queries
  • Ejercicio2: Index
  • Ejercicio2: Planes
  • Ejercicio2: Galería

Guía5

El contenido presentado en la quinta guía trata acerca de la introducción a JavaScript. En este se observa algunos usos que se le pueda dar este lenguaje de programación en las páginas web.
